El programa solicita al usuario el nombre de un Pokémon, busca su información en la API y muestra los detalles relevantes, como la imagen, peso, altura, movimientos, habilidades y tipos del Pokémon. Luego, guarda la información del Pokémon en un archivo JSON dentro de la carpeta "pokedex".  Se agrega un ciclo principal while True que permite repetir la búsqueda de Pokémon hasta que el usuario decida salir del programa ingresando "q". Dentro del ciclo, se solicita al usuario el nombre del Pokémon y se verifica si desea salir. Si el usuario ingresa "q", el ciclo se rompe y el programa termina. De lo contrario, se busca el Pokémon y se muestra su información.
